# To-do App

A simple todo app to help you what you want to do.



### Tech stack

* HTML
* CSS
* JavaScript



### **Features**

* Add new todo.
* Filter your todos.
* Mark as completed todo.
* Hide completed todos.
* Remove a todo.
* Count of all todos you have.
* Auto store your todos after done (Local Storage).
